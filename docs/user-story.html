WiseTraveler User Stories

For Travelers

  • As a traveler, I want to create an account with my email and a secure password so I can access personalized travel features.
  • As a traveler, I want to be prevented from registering with an email already in use so that my account remains unique.
  • As a traveler, I want to receive feedback if my registration fails due to errors like invalid email or password format.
  • As a traveler, I want to log into the system using my email and password so that I can access my profile and trip details.
  • As a traveler, I want to be shown an error message when login fails so that I know how to fix it.
  • As a traveler, I want to reset my password via a “Forgot Password” link so I can regain access to my account.
  • As a traveler, I want to receive a secure reset link by email when I forget my password.
  • As a traveler, I want to change my password from my profile for improved account security.
  • As a traveler, I want to view and update my personal information (e.g. name, date of birth, travel preferences) so that my profile stays accurate.
  • As a traveler, I want my updates to save in real-time and get confirmation so I know the changes were successful.
  • As a traveler, I want to see an engaging landing page when I visit the WiseTraveler site so I can explore its features.
  • As a traveler, I want to use links on the landing page to easily navigate to important features like registration, profile, or trip planning.
  • As a traveler, I want to view and agree to the Terms and Conditions before signing up so I understand my rights and responsibilities.
  • As a traveler, I want to take a short quiz to get destination recommendations that match my preferences.
  • As a traveler, I want to input preferences like budget, climate, activities, and duration in the quiz.
  • As a traveler, I want to get destination suggestions based on my responses.
  • As a traveler, I want to retake the quiz if my preferences change.
  • As a traveler, I want to ask travel-related questions to an AI assistant so I can get helpful answers.
  • As a traveler, I want the assistant to provide destination insights, safety info, and personalized tips based on my profile.
  • As a traveler, I want the assistant to maintain the conversation's context within a session for smooth interaction.
  • As a traveler, I want to access the AI assistant from any page so help is always one click away.
  • As a traveler, I want to explore destinations using an interactive map with zoom and filter features.
  • As a traveler, I want the map to show key info like attractions, accommodations, and points of interest.
  • As a traveler, I want to filter map content based on category (e.g., food, hotels, landmarks).
  • As a traveler, I want to see alerts related to health and safety when I select a destination so I can travel responsibly.
  • As a traveler, I want to be informed of crime, disasters, and travel advisories in the area I'm researching.
  • As a traveler, I want to learn about local customs, etiquette, and laws for the destinations I'm visiting.
  • As a traveler, I want to avoid cultural faux pas by reviewing dos and don'ts.
  • As a traveler, I want to see info about holidays and cultural events to better plan my trip.
  • As a traveler, I want to rate and review places I visit so I can help others.
  • As a traveler, I want to see other travelers' reviews and ratings to help guide my own travel plans.
  • As a traveler, I want to create a trip itinerary with names, locations, and dates to keep my travel organized.
  • As a traveler, I want to use a calendar to view and manage trip events like tours, meals, or check-ins.
  • As a traveler, I want to edit events on my calendar so I can make changes to my plans easily.

For Admin

  • As an administrator, I want the system to support RESTful APIs so that I can ensure smooth communication between frontend and backend services.
  • As an administrator, I want to secure all sensitive API calls over HTTPS to protect user data.
  • As an administrator, I want to integrate with Supabase for authentication and data storage.
  • As an administrator, I want to integrate with the OpenAI API so the chatbot can answer user queries.
  • As an administrator, I want to integrate with the Google Maps API so users can interact with map features.
  • As an administrator, I want the system to perform key functions quickly (e.g., loading landing pages in under 3 seconds) to enhance the user experience.
  • As an administrator, I want the application to support at least 1000 users concurrently without performance degradation.
  • As an administrator, I want database queries to return results within 10 seconds for efficient backend processing.
  • As an administrator, I want to use PostgreSQL (via Supabase) as the primary database to ensure consistency and scalability of data.
